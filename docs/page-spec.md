# 视图细化规格

## 目标
这份文档用于把 `视图地图` 继续收敛成可以直接进入线框设计和桌面渲染层实现的视图规格。  
重点覆盖：
- 启动台和工作台的模块层级
- 视图状态
- 关键交互
- 桌面端优先布局

MVP 以桌面端体验为主，不设计浏览器形态和移动端作为第一阶段目标。

## 1. 桌面启动台规格
### 视图定位
桌面启动台是 `知识宇宙总览视图`，不是普通列表页。  
它需要同时承担应用启动入口、知识库分发视图和知识状态总览视图三种职责。

### 桌面端视图结构
建议按以下顺序布局：

1. 顶部栏
2. 入口区
3. 继续探索便利贴
4. 知识墙
5. Provider 快捷入口区

### 1.1 顶部栏
#### 模块
- Logo / 产品名
- 全局搜索入口
- 新建知识库按钮
- Provider 入口
- 设置入口

#### 布局
- 左：Logo / 产品名
- 中：搜索入口
- 右：新建按钮、Provider、设置

#### 交互
- 点击 Logo 回到启动台顶部
- 点击搜索打开全局搜索层
- 点击新建知识库打开创建视图或创建弹层
- 点击 Provider 进入 Provider 管理视图

### 1.2 入口区
#### 模块
- 主按钮 `开始`
- 次按钮 `继续`

#### 规则
- 不放入口说明文案
- 不在首屏堆叠统计卡组和复杂总览图
- 入口区只保留动作，不承担产品介绍职责

### 1.3 继续探索便利贴
#### 模块
- 区块标题
- 最近知识库标题
- 一句短说明
- 节点数 / 更新时间
- 进入动作

### 1.4 知识墙
#### 模块
- 区块标题
- 便利贴式知识库网格

#### 规则
- 每张知识库卡片应像贴在墙上的知识块
- 只显示名称、节点数、更新时间和进入动作
- 允许轻微旋转和色差，但不能造成阅读混乱

### 1.5 Provider 快捷入口
#### 模块
- 当前默认 Provider
- 连接状态
- 进入 Provider 管理视图

## 2. 知识工作台规格
### 视图定位
知识工作台是系统的主操作面。  
所有高频节点操作默认都应在这里完成，不再拆出额外视图。

### 桌面端视图结构
建议采用固定五区结构：

1. 顶部栏
2. 左栏结构导航
3. 中央图谱画布
4. 右栏节点详情
5. 底部输入区

### 2.1 顶部栏
#### 模块
- 返回启动台按钮
- Provider 状态指示
- 当前知识库名称
- 搜索入口
- 知识库设置入口
- 当前节点快捷操作

### 2.2 左栏结构导航
#### 模块
- 当前节点祖先路径
- 直接子节点列表
- 最近访问节点

### 2.3 中央图谱画布
#### 模块
- 图谱画布
- 极简图例

### 2.4 右栏节点详情
#### 模块
- 标题区
- 摘要区
- 正文区
- 来源信息区
- 相关节点区
- AI 建议区

#### 规则
- 右栏默认先展示阅读态，不默认进入表单编辑态
- 编辑作为次级动作，通过 `编辑` 按钮切换
- 上下文、来源和主父节点信息只保留必要字段，不补说明句

### 2.5 底部输入区
#### 模块
- 输入框
- 当前上下文提示
- 模式切换
- `候选` 按钮
- `长出` 按钮

## 3. 辅助层规格
### 3.1 候选节点确认层
#### 形式
- 优先以内嵌在右栏的紧凑卡片出现
- 数量过多时再升级为抽屉

### 3.2 全局搜索层
#### 形式
- 居中浮层

#### 交互
- 在启动台可搜索知识库和节点
- 在工作台内优先搜索当前知识库节点

### 3.3 节点结构编辑层
#### 形式
- 右栏编辑态 + 辅助弹层

### 3.4 Provider 管理视图
#### 模块
- Provider 列表
- Provider 表单
- 连接测试按钮
- 默认模型绑定
- 错误状态提示

#### 布局
- 左侧使用紧凑列表展示已接入 Provider
- 右侧为单张安静表单面板
- 不做厚重后台感分页和表格

#### Provider 表单字段
- 名称
- Base URL
- API Key
- Default Model
- Enabled

#### 规则
- 仅支持 OpenAI-compatible 配置格式
- 不根据不同厂商切换字段集

## 4. 视图状态清单
### 桌面启动台
- 空状态
- 初次使用状态
- 常规状态
- 搜索筛选状态

### 知识工作台
- 根节点状态
- 普通节点状态
- 无子节点状态
- 候选生成中状态
- 节点编辑状态
- Provider 未配置状态

## 5. 桌面端尺寸建议
### 启动台
- 顶部栏：64px 到 72px
- 入口区：单块大卡片，不追求占满整屏
- 知识墙最大内容宽度：1200px 到 1440px

### 工作台
- 顶部栏：56px 到 64px
- 左栏：280px 到 320px
- 中栏：自适应主区域
- 右栏：360px 到 420px
- 底部输入区：固定悬浮或吸底

## 6. MVP 不做的视觉和交互
- 启动台总览区自由拖拽排布
- 工作台中的复杂协同编辑
- 多窗口图谱比较
- 独立浏览器形态适配
- 启动台和工作台中高成本 3D 动效

## 7. 实现优先级
### P0
- 启动台入口区
- 启动台知识墙
- 工作台五区结构
- Provider 管理视图
- 候选节点确认层

### P1
- 启动台继续探索便利贴
- 节点结构编辑层

### P2
- 更丰富的启动台状态标签
- 更复杂的知识墙编排
- 图谱全览页
